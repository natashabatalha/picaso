PICASO tutorials are now provided as Jupytext .py files. To learn how to download and run these as notebooks, please see our Tutorial Workflow page.

Getting Data and Setting up Environments after Installation

This quickstart notebook has some overlap with our installation instructions (Please read the installation page first if you are unfamiliar with package installation, environment variables, etc), but it is designed to be a quick way to get up and running with PICASO after you have installed it. It includes instructions for getting reference data, setting up environments, and checking environments have been successfully setup. .. toctree:
