One-Dimensional Climate Models: Brown Dwarfs w/ Disequilibrium Chemistry with Constant Kzz (~Elf Owl)ÔÉÅ

In this tutorial you will learn how to run 1d climate models with the effects of disequilibrium chemistry as was done for the Elf-OWL Grid Mukherjee et al. 2024 (note this should also be cited along with `PICASO 4.0 <>`__ if using this code/tutorial).

  • basics of running/analyzing thermal spectra

  • how to analyze thermal emission spectra

  • how to run a basic 1d brown dwarf tutorial
